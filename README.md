# azure-communication-services-domain-checker
Single‑file PowerShell web UI + REST API to validate Azure Communication Services email domain readiness (TXT/SPF, MX, DMARC, DKIM, CNAME, DNSBL reputation, WHOIS/RDAP). Includes optional API key auth, rate limits, anonymous metrics, Entra ID sign‑in, and Docker‑ready deployment.
